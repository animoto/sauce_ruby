# Sauce::Cucumber

#[DEPRECATED] Sauce Labs does not recommend this gem. It will not receive further development or support.

## Installation

Add this line to your application's Gemfile:

    gem 'sauce-cucumber'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sauce-cucumber
